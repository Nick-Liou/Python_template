# Python_template
A template for Python projects with github actions for 
- tests (with pytest)
- type anotations (with Mypy)



## Installation

    ```

    ```

    ```
    cd NAME
    ```
3. Install the required dependencies using pip:
    ```
    pip install -r requirements.txt
    ```
